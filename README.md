# mynodejs
first nodejs app with restful api, mysql, jwt

First, install dependencies using npm:

```sh
npm install
npm install -g mocha

```
Then, run the app:

```sh
node ./bin/www
```

and use ./test/restfultest.js to test the api

```sh
mocha ./test/restfultest.js
```
